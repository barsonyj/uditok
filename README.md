# Üdítők
A kedvenc üdítőink képét, nevét és méretét tudjuk benne összegyűjteni és megjeleníteni.
### Telepítés
Ha szeretnéd kipróbálni, add ki a következő parancsokat:
```
git clone https://github.com/barsonyj/uditok
cd uditok
pnpm i
pnpm dev
```
